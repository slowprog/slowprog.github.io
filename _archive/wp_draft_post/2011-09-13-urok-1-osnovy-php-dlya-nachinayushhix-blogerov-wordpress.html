</a>Доброе время суток, дорогой читатель. Не для кого не секрет, что многие, кто ведет блоги на WordPress не понимают в PHP или даже плохо себе представляют, что это такое. Как правило, это не очень хорошо, т.к. блогер сильно ограничен: он не может изменять код шаблона или плагина под свои нужды, не может писать с нуля специфические плагины для определенных целей, не может диагностировать и исправлять ошибки в случаи их возникновения и другое. Это все очень печально. И я подумал, что стоит сделать цикл уроков для начинающих программистов PHP, притом не обычных уроков, а заточенных именно под WordPress, чтобы в дальнейшем такие блогеры не терялись при виде кода, а как минимум могли его прочитать и понять. Так как уроки будут заточены под WordPress, то мы пропустим много, что нам для начала не понадобится. В последствии мы это наверстаем, если будет необходимость.

Основы</h2>
Для начала давайте разберем основные понятия и рассмотрим основные конструкции, чтобы в дальнейшем не было вопросов, и мы могли без вопросов ими оперировать.

Переменная </strong>— это именованная область компьютерной памяти, которую применяют для записи, хранения, доступа и использования каких-либо данных.</blockquote>
В РНР переменные начинаются со знака доллара ($), за которым следует любое количество букв, цифр и символов подчеркивания, но первый символ не может быть цифрой. Так же стоит учитывать, что переменные чувствительны к регистру, т.е. переменные $foo</em> и $Foo</em> являются различными.

Типы данных</strong> в PHP являются неявными (динамические) и потому объявлять переменную перед её использование не обязательно.</blockquote>
Переменные в PHP не требуют явного объявления и указания типа, это все происходит неявно при первом ее применении, и при этом одна и та же переменная может иметь на протяжении работы кода разные типы, т.е. с одной и той же переменной на протяжении кода программы можно работать как со строкой, и как с числом. Однако, в РНР все таки существует набор основных типов данных:

  • integer</strong> - целочисленный тип данных, который служит для представления целых чисел (от –2 147 483 648 до 2 147 483 647).</li>
  • float</strong> - число с плавающей запятой, которые используются для представления дробных чисел (максимум 1.8e308 с точностью около 14 десятичных цифр после запятой).</li>
  • string</strong> - тип хранящий набор символов любой длины, т.е. используемый для хранения строк.</li>
  • boolean</strong> - простейший тип для выражения истинности значения, который может содержать всего два значения: либо правда (TRUE</em>), либо ложь (FALSE</em>).</li>
  • array</strong> - упорядоченный набор однотипных переменных, доступ к которым осуществляется по индексу (номеру по порядку, например, foo[4]</em>).</li>
  • object</strong> - объектный тип данных определяет некоторый объект, который имеет определённое состояние и поведение, другими словами, обладает свойствами (атрибуты) и операциями над ними (методы). Но вы пока особо об этом не думайте, этим оперировать не будем пока.</li>
    </ul>

    Оператор</strong> — это наименьшая автономная часть языка программирования, которая выполняет какое-либо одно элементарное действие.</blockquote>
    Любые действия в программировании осуществляются при помощи элементарных операторов, и любую функцию можно разложить на эти операторы. В конце каждого оператора ставиться точка с запятой (;). Она обозначает конец оператора и является обязательной для постановки, иначе PHP будет ругаться. В PHP достаточно много различных операторов, но для начала, мы с вами будем использовать только некоторые из них. Начнем с основных операторов:

    • Присвоение</strong> (=)</li>
    • Конкатенация</strong> (.)</li>
    • Сложение</strong> (+)</li>
    • Вычитание</strong> (-)</li>
    • Умножение</strong> (*)</li>
    • Деление</strong> (/)</li>
      </ul>
      Оператор присвоения используется, когда необходимо какой-либо переменной присвоить новое значение. Присвоенное значение может быть любого допустимого типа данных, о которых говорилось выше. Операция конкатенации применяется только для строковых переменных (но вы можете не париться об этом, т.к. тут неявное преобразование типов), с помощью нее склеивают две строки в одну. Следующие четыре оператора, я думаю, не составят труда для понимания, т.к. это проходилось в школе. Все эти операторы могут быть легко проиллюстрированы:
      [cce lang="php"]$i = 4;
      $foo = (2 + $i) * 2 / 4 - 2;
      echo 'Результат равен '.$foo;[/cce]
      Вначале переменной $i</em> присваиваем значение 4</em> и после переменной $foo</em> присваиваем результат работы всех арифметических операторов. И как видно из примера, переменные могут использоваться как слева, так и справа от знака присваивания. В третьей строке используется оператор конкатенации для склейки строки 'Результат равен '</em> и значения переменной $foo</em>. И эта склеенная строка выводится на экран с помощью оператора echo</em> (я его не стал выделять отдельно).

      Оператор ветвления if-else</h2>
      Следующий оператор который мы рассмотрим будет if-else</em> оператор условного выбора, который действует следующим образом: оператор if(condition)</em> проверяет условие condition</em>, если его значение истинное (TRUE), то выполняется код идущий после if</em>, если условие ложно (FALSE) – выполняется код после оператора else</em>, если он присутствует (т.к. else не обязательный элемент). Для того, чтобы использовать этот оператор необходимо уметь пользоваться операторами сравнения, которые служат для сравнения двух значений:

      • Равно</strong> (==) — TRUE если два значения равны.</li>
      • Тождественно равно</strong> (===) — TRUE если значения равны и имеют один и тот же тип.</li>
      • Не равно</strong> (!= или <>) — TRUE если значение не равны.</li>
      • Тождественно не равно</strong> (!==) — TRUE если значения не равны или разных типов.</li>
      • Меньше</strong> (<) — TRUE если первое значение строго меньше второго.</li>
      • Больше</strong> (>) — TRUE если первое значение строго больше второго.</li>
      • Меньше или равно</strong> (<=) — TRUE если первое значение меньше или равно второго.</li>
      • Больше или равно</strong> (>=) — TRUE если первое значение больше или равно второго.</li>
        </ul>
        C помощью этих операторов сравнения можно производить сравнения любых типов данных, как числовых, так и строковых, и объектных. Теперь можно рассмотреть пример использования оператора if</em>. Сразу скажу, что фигурные скобочки обрамляют исполняемый код, который будет выполняться если условие выше истинно. Пример будем рассматривать сразу сложный, чтобы разобрать все тонкости его работы:
        [cce lang="php"]$foo = "asdf";
        if (($foo == "asdff") or ($foo != "99"))
        {
        echo "$foo равно asdff и не является 99";
        }
        elseif ($foo != "")
        {
        echo "$foo не пустое";
        }
        else
        {
        echo "Что-то другое";
        }[/cce]
        Как вы можете заметить, появились новые элементы в конструкции оператора. Элемент elseif</em> обозначает еще одно условие (ветвление), которое может выполняться или нет. Другими словами, если первый if</em>  (условие) не выполняется, то проверяется второе условие. А если и второе условие оказывается ложным, то выполняется код расположенный в теле else</em>. Как вы понимаете, elseif</em> можно использовать сколько угодно раз, а else</em>, все так же, не является обязательной частью и может быть опущено. То есть, может быть такое, что ни одно elseif</em> не будет истинно и если в конце нет else</em>, то весь код оператора if</em> пропускается.

        Следующее, что может вам показаться новым это or</em> в первом условии. Это логическое ИЛИ, оно является одним из логических операторов. Самые распространенные логические операторы это следующие:

        • Логическое ИЛИ</strong> (or или ||).</li>
        • Логическое И</strong> (and или &&).</li>
        • Логическое отрицание НЕ</strong> (!).</li>
          </ul>
          Они используются для построения сложных логических условий в операторах ветвления типа if</em>. Первые два, скорее всего, у вас не вызовут затруднения, а вот третий я проиллюстрирую. Логическое отрицание ставится перед логическим выражением и меняет его значение на противоположное, если бы его не было. Например, если к оператору сравнения =</em> (равно) подставить логическое отрицание, то получится противоположное значение !=</em> (не равно).

          Теперь вы сможете свободно прочитать предыдущий пример кода и понять его смыл. Но чтобы не возникло разногласий я привожу этот же пример код, но с комментариями (комментарии это все что идет после //</em>, комментарии PHP не замечает, туда можно писать, что угодно):
          [cce lang="php"]// Присваиваем значение asdf переменной $foo
          $foo = "asdf";
          // Если $foo равно asdff или не равно 99, то выполняем код ниже
          if (($foo == "asdff") or ($foo != "99"))
          {
          echo "$foo равно asdff и не является 99";
          }
          // Если $foo не равно пустой строке (две кавычки обозначают пустую строку)
          elseif ($foo != "")
          {
          echo "$foo не пустое";
          }
          // Если не одно условие не подошло то выполняем код ниже
          else
          {
          echo "Что-то другое";
          }[/cce]
          Вот теперь надеюсь всем всё понятно. На сегодня новых элементов хватит. Надеюсь вы можете уже понимать простой код. Теперь давайте сделаем что-нибудь полезное с помощью этих знаний.

          Простая функция на PHP</h2>
          Для лучшего усвоения материала, я думаю, стоит написать рабочий код, который вам пригодится в блогах на WordPress. Я решил, что это будут хлебные крошки. Для тех кто не знает - это путь пройденный вами по сайту от главной страницы. Достаточно полезная вещь, как для пользователей, так и для поисковых систем при индексации. Подразумевается, что с HTML и CSS у вас проблем нет и ничего объяснять не придется. Иначе советую вначале их подтянуть.

          Эту функцию пишем в специальном файле вашей темы под название functions.php</em>, который вы можете редактировать либо через FTP-доступ, либо прямо в WordPress во вкладке Внешний вид</em> -> Редактор</em>.

          В коде вы увидите много новых функций. В основном это функции самого WordPress, прочитать по подробнее про которые вы сможете на codex.wordpress.org</a>. Но для того, чтобы код был более понятный я каждую строчку прокомментирую:
          [cce lang="php"]// Функция вывода хлебных крошек
          function add_breadcrumb()
          {
          // Открываем список
          echo '

            ';
            // Если это страница не главная, то
            if (!is_home())
            {
            // выводим ссылку на главную страницу
            echo '
          • Главная</a></li>';
            // Если это категория или одиночная запись, то
