    Переадресация Blogger на WordPress</h2>
    Чтобы WordPress мог перекидывать на нужную страницу надо немного покодить. Необходимо в папке с вашей активной темой разместить файл, который будет принимать трафик с Blogger и перенаправлять на нужную страницу. Скопируйте ниже приведенный код и вставьте его в файл с именем blogger-redirection.php</em>, который необходимо разместить в wp-content/themes/your_active_theme</em>, где your_active_theme</em> - это имя папки темы, которую вы используете в данный момент в WordPress (и обратите внимание, что если вы смените тему, то необходимо скопировать этот файл и в неё):

    [cce lang="php"]
    

/*
    
	Template Name: Blogger Redirection
    
*/
    

    	global $wpdb;
    
	$old_url = $_GET['q'];
    

    	if ($old_url != "") {
    
		$permalink = explode("blogspot.com", $old_url);
    

    		$q = "SELECT guid FROM $wpdb->posts LEFT JOIN $wpdb->postmeta ".
    
		        "ON ($wpdb->posts.ID = $wpdb->postmeta.post_id) WHERE ".
    
		        "$wpdb->postmeta.meta_key='blogger_permalink' AND ".
    
		        "$wpdb->postmeta.meta_value='$permalink[1]'";
    

    		$new_url = $wpdb->get_var($q)? $wpdb->get_var($q) : "/";
    

    		header ("HTTP/1.1 301 Moved Permanently");
    
		header("Location: $new_url");
    
	}
    
?>
    
[/cce]</pre>
    
Теперь у нас есть файл, который является индивидуальным шаблоном для нашей страницы перенаправления. Далее, необходимо создать новую страницу в WordPress, которую нужно назвать blogger</em> (это важно) и выбрать индивидуальный шаблон, что мы сделали. Выбрать шаблон можно в виджете Атрибуты страницы</em> в выпадающем списке Шаблон</em>, нам необходим шаблон под названием Blogger Redirection</em>. Сохраните страницу. Теперь WordPress готов к приему редиректов с Blogger, и осталось внести некоторые изменения в сам Blogger для организации редиректов на ваш сайт WordPress.
    

    Войдите в свой аккаунт Blogger и в панели управления вашим блогом перейдите в Шаблон</em>. Там пролистайте в самый низ страницу и нажмите Вернуться к классическим шаблонам</em>, вам откроется предупреждение о том что этого лучше не делать, но вы не взирая ни на что нажмите снова Вернуться к классическому шаблону</em>, подтверждая тем самым свои намерения. Затем в поле с кодом шаблона скопируйте (вместо того, который есть там) нижеприведенный код, заменив все http://www.yourdomain.com</em> на ваш новый домен для WordPress (обязательно, чтобы вначале домена было http://</em>).
