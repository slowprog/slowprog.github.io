Вначале немного теории. Обычно используется иконка</strong> 16×16 пикселей с названием favicon.ico, ранее ее надо было обязательно помещать в корень сайта, сейчас же, в случае, если в коде страницы явно не указан адрес иконки, то браузер пытается загрузить favicon.ico из корня сайта автоматически. Однако держать фавикон в корне сайта не обязательно, имеется возможность указать положение иконки, что позволяет при условии отсутствия изображения в корне использовать свой собственный значок для каждой страницы. Чтобы указать для страницы ее фавикон, необходимо поместить в код вашего сайта внутрь тега head</em> строку следующего вида:
[cce lang="xml"] [/cce]
Значение rel</em> обычно содержит «shortcut icon</em>», это необходимо, чтобы IE  реагировал на всю строку (как всегда у парней все через одно место), а нормальные браузеры — на слово «icon</em>». Значение параметра href</em> содержит URI к изображению. Соответственно, type</em> содержит формат файла. Он может быть png или gif, размером 16x16 или 32x32, глубина цвета может быть 8-битной или 24-битной.

Для того, чтобы поместить иконку для блога на WordPress</strong> необходимо для начала ее сделать. Вначале сделаем любую картинку в любом редакторе (я предпочитаю Photoshop), к ней будут следующие требования:

  • Картинка должна быть квадратной, иначе все будет малость кривое.</li>
  • Необходимо сохранить в форматах gif, jpg, png и bmp.</li>
  • Прозрачность</strong> может быть использована только в форматах gif, png и ico.</li>
  • Максимальный размер картинки 150.00 kB.</li>
    </ul>
