</a>Как вы думаете, красивые кнопки с помощью CSS</strong>, которые будут выглядеть в самом деле хорошо - это миф? Лично я думаю, что это далеко не миф и с помощью CSS можно создать очень симпатичные и привлекательные кнопки. Ведь иногда так лениво делать сплайны для новых кнопок, пытаться что-то выдумать и копошиться в Фотошопе, что лучше накидать быстренько кнопку в CSS и расслабится. Вот именно для таких целей я и покажу вам, как это сделать совершенно бесплатно.

Но перед этим немного порассуждаю и опишу пару положительных сторон. Использование только голого CSS для создания кнопок является хорошим выходом и оптимальным решение во многих ситуациях, если конечно у вас не уникальный дизайн требующий уникальных элементов. Во-первых, сразу отпадает необходимость подгружать лишние картинки (сплайны) для кнопок, что ускоряет загрузку контента, тем более если у вас куча разных кнопок и под каждую свой сплайн. Во-вторых, на лицо экономия времени при разработке дизайна. Ведь куда быстрее вставить стили, чем придумывать и рисовать в Фотошопе кнопочки. А в-третьих, в части случаях дизайн с помощью сплайнов может быть повторен с помощью CSS.

Теперь все убедились, что в некоторых случаях использовать CSS кнопки</strong> действительно полезно, и теперь можно приступить к созданию этих самых стилей.

Динамика</h2>
Добавим немного динамики нашим кнопкам, ведь в использовании сплайнов в том и соль, что они мигают и переливаются всеми цветами радуги когда их гладишь и кликаешь. А мы чем хуже? Сделаем так же! Я приведу пример только для первой кнопки, но он будет верен и для всех остальных, просто их уже придется написать вам.

Чтобы наша кнопка ожила, мы используем псевдоклассы CSS, а точнее два из них: hover и active. Первый отвечает за наведение, а второй за активацию объекта, в данном случаи за клик. Пусть когда мы будем наводить мышкой на кнопку она просто поменяет цвет, а когда кликнем она вожмется, т.е. внешнюю тень переделаем на внутреннюю.

[cce lang="css"].cool-button-1:hover {
background-color: #FA8;
}

.cool-button-1:active {
background-color: #FA8;
box-shadow: inset rgba(0, 0, 0, 0.3) 1px 1px 0px 0px;
-webkit-box-shadow: inset rgba(0, 0, 0, 0.3) 1px 1px 0px 0px;
-moz-box-shadow: inset rgba(0, 0, 0, 0.3) 1px 1px 0px 0px;
-o-box-shadow: inset rgba(0, 0, 0, 0.3) 1px 1px 0px 0px;
-ms-box-shadow: inset rgba(0, 0, 0, 0.3) 1px 1px 0px 0px;
}[/cce]
Все достаточно просто и элементарно. А теперь подумайте, как долго пришлось бы рисовать это в Фотошопе, а потом писать стили. Жуть просто! Не усложняйте себе жизнь.

Градиент</h2>
C ловкой подачи Сергея</strong> в комментариях, я решил дополнить этот пост примером использования градиента в кнопках, точнее в фоне кнопок. Это конечно целая огромная отдельная тема, но для новичков пока дам просто пример, чтобы они знали, что такое делать можно. Позже я может напишу отдельный пост про организацию градиентов.

Сделаем простой линейный кроссбраузерный градиент, на фоне кнопки №1 при наведении мышкой вместо имеющегося сейчас:

[cce lang="css"].cool-button-1:hover {
background: #FFAA22;
filter: progid:DXImageTransform.Microsoft.gradient(startColorstr='#FFAA22', endColorstr='#FFD48F');
background: -webkit-gradient(linear, 0% 0%, 0% 100%, from(#FFAA22), to(#FFD48F));
background: -webkit-linear-gradient(top, bottom, #FFAA22, #FFD48F);
background: -moz-linear-gradient(top, #FFAA22, #FFD48F);
background: -o-linear-gradient(top, #FFAA22, #FFD48F);
background: -ms-linear-gradient(top, #FFAA22, #FFD48F);
}[/cce]
Первая строчка для подстраховки, на тот случай если браузер не поддерживает CSS3 он просто будет отображать стандартный фон.

Как мне подсказали в комментариях, в IE9 при наведении на кнопку отваливается радиус и все углы "бессовестно выпучиваются" (цитата). Это все потому что IE9 не переваривает одновременное использование радиуса и градиента, такой уж он. Если честно, то мой совет вообще не использовать их вместе. Но это какой-то несуразный совет, да, и есть другой выход - использовать SVG</a>. Вместо кода добавления градиентов вам надо добавить следующее:
