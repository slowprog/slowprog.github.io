            • $category</em> — булеановская переменная отвечающая за вывод блока с написанными постами. Если true</em> — выводить блок с постами, false</em> — не выводить блок с постами.</li>

            • $categoryExclude</em> — переменная отвечающая за исключение определенных категорий из списка категорий при выводе постов. Т.е. все посты их исключенных категорий выведены не будут.             • $categoryEmpty </em> — булеановская переменная отвечающая за вывод пустых категорий. Если true</em> — пустые категории не будут выведены, false</em> — пустые категории будут выведены.</li>
            • $page</em> — булеановская переменная отвечающая за вывод блока со страницами. Если true</em> — выводить блок, false</em> — не выводить блок.</li>
            • $pageExclude</em> — переменная отвечающая за исключение определенных страниц из списка страниц. Для исключения нужных страниц необходимо их номера перечислить через запятую, в противном случаи оставить переменную пустой, т.е. присвоить ей пустую строку ($pageExclude= ''</em>).</li>
            • $author</em> — булеановская переменная отвечающая за вывод блока с авторами блога. Авторы - это пользователи которые могут писать и опубликовывать статьи в блог. Если true</em> — исключить, false</em> — не исключить.</li>

              [cce lang="php"][/cce]
